A Ruby implementation of D-ary Huffman Encoding and Decoding. This program is used to generate genetic 
sequences and encode them, the genetic sequence can also be decoded.

**********************************************
*****RUBY MUST BE INSTALLED ON THE MACHINE****
**********************************************
To run code questions open a terminal and navigate 
to the HuffmanCoding directory. From there enter a
command(below) to run a specific question and follow 
the guidelines displayed in the terminal.

Commands for running each question from command line:
Q1 = "ruby q1.rb"
Q2A = "ruby q2_a.rb"
Q2B = "ruby q2_b.rb"
